# AutoTapper

AutoTapper 是一个 Android 自动连点工具。  
它通过无障碍服务执行点击手势，通过悬浮控制器在任意应用中完成取点、开始点击和停止点击。

这份 README 只保留给用户看的说明书。

## 适合做什么

- 在一个固定位置连续点击
- 在任意应用中快速开始或停止自动点击
- 临时重新调整点击点位
- 用简单配置完成基础自动点击

## 当前支持

- 单点连续点击
- 悬浮控制器常驻侧边
- `+` 添加或重选点击点
- `DEL` 删除当前点击点
- `GO` 开始点击
- `STOP` 停止点击
- 半透明橘色点位标记
- 基础间隔
- 随机附加延迟
- 固定次数 / 无限次数

## 当前不支持

- 多点轮流点击
- 滑动手势
- 复杂动作编排

## 首次使用

1. 安装并打开 AutoTapper。
2. 点击“授予悬浮窗权限”。
3. 点击“开启无障碍服务”，并在系统设置中启用 AutoTapper。
4. 回到应用，设置点击配置。
5. 点击“打开悬浮控制器”。

如果当前还没有点击点位，应用会直接进入取点模式。

## 基本使用流程

1. 打开悬浮控制器。
2. 点击 `+`。
3. 在目标位置轻触一次，记录点击点位。
4. 屏幕上会出现一个半透明橘色圆点，表示当前点击位置。
5. 进入你真正需要操作的应用。
6. 展开侧边悬浮控制器。
7. 点击 `GO` 开始自动点击。
8. 需要停止时，点击 `STOP`。

## 悬浮控制器说明

悬浮控制器默认收在屏幕侧边。

- `+`：添加或重新选择点击点
- `DEL`：删除当前点击点
- `GO`：开始自动点击
- `STOP`：停止自动点击

当控制器展开时：

- 如果已经设置了点击点，会显示一个半透明橘色圆点
- 点击进行中时，这个圆点会闪烁

当控制器收起时：

- 点位标记会自动隐藏
- 不会一直挡住屏幕内容

## 点击配置说明

### 基础间隔

表示两次点击之间至少等待多久，单位是毫秒。

- `1000` 毫秒 = `1` 秒
- 建议先从 `300` 到 `800` 之间尝试
- 如果填写小于 `100`，会按 `100` 毫秒处理

### 随机附加延迟

表示在基础间隔之外，再额外随机增加一点等待时间。

例如：

- 基础间隔 = `400`
- 随机附加延迟 = `80`

那么每次点击后的实际等待时间，会在 `400` 到 `480` 毫秒之间变化。

如果填 `0`，表示固定节奏点击，不加入随机变化。

### 点击次数

- 填 `1`：只点击 1 次
- 填 `10`：点击 10 次
- 填 `0`：一直点击，直到手动停止

## 使用建议

- 第一次使用时，先在不重要的界面测试参数
- 如果点位有偏差，直接用 `DEL` 删除，再用 `+` 重新取点
- 如果点击太快，先把基础间隔调高
- 如果想让节奏不那么死板，可以适当增加随机附加延迟

## 使用提醒

- AutoTapper 不会因为你开启了无障碍服务就自动点击
- 只有你手动点击 `GO` 或主界面的“开始点击”时，才会真正执行
- 请只在你明确知情并允许的场景下使用
- 不建议用于违反平台规则、规避风控或影响他人服务的行为
